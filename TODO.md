## Task tracking (bd)

`monorepo/` work is tracked with **bd (beads)**.

- **Ready work**: `bd ready --json`
- **Create**: `bd create "Title" -t bug|feature|task|chore|epic -p 0-4 --json`
- **Update**: `bd update <id> --status in_progress|open|blocked --json`
- **Close**: `bd close <id> --reason "Done" --json`
